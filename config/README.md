# Alumni Portal - PHP

A web-based Alumni Portal connecting Students, Alumni, and Admin.

## Live Demo
http://alumni-portal-php.infinityfreeapp.com/alumni_portal_php_enhanced

## Tech Stack
- PHP (Server-side)
- MySQL (Database)
- HTML/CSS (Frontend)

## Features
- 3 Role Login: Student, Alumni, Admin
- Post Jobs, Events, Messages
- Profile Management
- File Uploads

## Setup (Local)
1. Install XAMPP
2. Copy `config/db.sample.php` → `config/db.php`
3. Update DB credentials
4. Import `alumni_portal_infinityfree.sql` in phpMyAdmin
5. Run via `localhost/alumni_portal_php_enhanced`
